Sobre o Projeto

O que é um Servidor Web?

Um servidor web é um software e hardware que usa HTTP (Hypertext Transfer Protocol) e outros protocolos para responder a solicitações de clientes feitas pela World Wide Web.

O que é IIS?

O Internet Information Services (IIS) é um servidor web flexível, seguro e gerenciável da Microsoft para uso com o Windows NT.

O que é Windows Server?

Windows Server é um sistema operacional de servidor da Microsoft, projetado para compartilhamento de serviços de dados, aplicativos e recursos de rede.

Hospedagem de Sites

Hospedagem de sites é o serviço que permite que indivíduos e organizações tornem seus sites acessíveis via a World Wide Web.

HTTP, HTTPS e TCP/IP

HTTP é o protocolo de transferência de hipertexto, HTTPS é a versão segura, e TCP/IP é o conjunto de protocolos que permite a comunicação na internet.

Hyper-V

Hyper-V é a tecnologia de virtualização da Microsoft que permite criar e gerenciar máquinas virtuais no Windows Server.
